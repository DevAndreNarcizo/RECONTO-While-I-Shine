# Pasta de Assets

Coloque aqui os arquivos de arte/som conforme você for produzindo. Subpastas:

- `sprites/` — personagens (lendas), inimigos, bosses, projéteis, efeitos, tilesets.
- `audio/` — crie `musica/` e `sfx/` dentro.
- `ui/` — botões, ícones, fontes, HUD.
- `backgrounds/` — fundos de menu e telas.

> Quando criar o projeto Godot de verdade, esta pasta `assets/` vira `res://assets/` dentro do projeto.
> Aqui em `/var/www/Jogo-2D/assets/` ela serve como staging/organização enquanto você reúne o material.

⚠️ **Registre a licença de TODO asset de terceiros** em `../referencias/licencas-assets.md` antes de usar
em build comercial. Ver `prompts/sprites-arte/07-itch-io-guia.md` §B.
